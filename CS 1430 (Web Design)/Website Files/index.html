Class Reflection

During my time in CS 1430 I learned a ton about how to create not just from the perspective of a programer but visually and artistically as well. I found a lot of interest in how I was able to manipulate code to do what I wanted it to do in ways I had not thought of while taking all my standard coding classes with Java, and began to find a more artistic side to coding as I explored new and more creative ways to finish assingments. I enjoyed how simple and entertaining it was to code in HTML, having a language with simple rules really made for more time for me to focus on more exciting elements like CSS and JavaScript. With more lectures and more tools in my belt I began to see the websites and other web-based programs with more interest and would dig into them to see what kinds of code they had working behind the screen. This led me to reach out to other professional website designers to learn more about how I can expand my abilities and further garner inspiration to develope better websites.
